# Проект интернет-магазина цифровых товаров на Django.

## Описание

Это веб-приложение для продажи плагинов, скриптов и других цифровых товаров.

## Технологии

- Python 3.13
- Django 5.0
- Poetry
- Bootstrap 5

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `poetry install`
3. Активировать окружение: `eval $(poetry env activate)`
4. Выполнить миграции: `python manage.py migrate`
5. Запустить сервер: `python manage.py runserver`
